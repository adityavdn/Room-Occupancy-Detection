# 🏠 Room Occupancy Detection

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![IoT](https://img.shields.io/badge/IoT-Smart_Home-blueviolet.svg)

## 📖 Overview

Binary classification model to detect room occupancy using environmental sensor data (temperature, humidity, light, CO2). Useful for smart building automation and energy optimization.

## 🎯 Features

- Sensor data analysis
- Real-time occupancy detection
- Energy efficiency insights
- Multiple classification algorithms

## 🛠️ Tech Stack

- Scikit-learn (Logistic Regression, Random Forest, SVM)
- Pandas, NumPy
- Matplotlib, Seaborn

## 🚀 Usage

```bash
git clone https://github.com/adityavdn/Room-Occupancy-Detection.git
pip install -r requirements.txt
jupyter notebook
```

## 📊 Performance

- Accuracy: 98%+
- Precision/Recall: Balanced
- Feature importance: CO2, Light levels
